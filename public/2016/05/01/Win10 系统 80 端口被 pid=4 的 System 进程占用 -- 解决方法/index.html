Win10 系统 80 端口被 pid=4 的 System 进程占用 -- 解决方法

发表于 2016-05-01   |     |   阅读次数

背景：

今天在家里电脑下载 phpstudy 2016 后，准备启动服务器，结果提示 80 端口被占用。

过程：

百度后有这几种解决方法：

1.IIS 占用


2.SQL server 2008 的报表服务占用

可惜这两种方法试了都不行。

解决：

1.打开注册表：regedit

2.找到：HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\HTTP

3.在右边找到 Start 这一项，将其改为 0

4.重启系统
