3.利用 PHP file_put_contents() 函数

（1）少用错误控制运算符”@”

PHP 支持一个错误控制运算符：@ 。当将其放置在一个 PHP 表达式之前，该表达式可能产生的任何错误信息都被忽略掉。

如果一个缺陷发生在这个表达式中，从PHP的输出中看不到任何错误，这增加了调试的难度。所以能不用则不用。

@ 运算符只对表达式有效。对新手来说一个简单的规则就是：如果能从某处得到值，就能在它前面加上 @ 运算符。
